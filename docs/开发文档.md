# 开发帮助
1. 命令行参数激活对应环境的配置：`--spring.profiles.active=dev_mem`
2. VM参数，是否运行在无GUI的服务器环境下，true为无GUI环境：`-Djava.awt.headless=false`
___

# 发布格式
1. tag和发布名称，使用小写字母v加版本号：GitHub的建议，以及SpringBoot的发布格式，都是这样做的。
2. 新发布版本包含什么内容就写对应部分内容，不包含不写。
```markdown
### 版本变化
1. 新增：功能1
2. 新增：功能2
3. 变更：xxx
4. 修复：问题1
5. 修复：问题2
6. 优化：xxx
### 升级注意
1. xxx
```
